# Search Flight API

Esta é uma API que retorna a escala de voos e lista de aeroportos.
Ela foi desenvolvida em Spring Boot com o gerenciamento de dependências em Maven.

## Configuração

Instale o Docker em seu computador [https://docs.docker.com/install](https://docs.docker.com/install)

## Execução

```bash
docker run -p 127.0.0.1:9090:9090/tcp -it gabrieljscunha/flights-application:latest
```
